# 更新日志

## [1.6.0] - 2026-03-11
### 新增
- **MCP 服务器正式版**：`memory_mcp_simple.py` 成为主服务，支持标准 MCP 工具调用。
- **开机自动加载**：通过 `simple_startup_loader.py` 在 MCP 启动时自动加载记忆系统，注入上下文。
- **工具管理系统**：`mechanisms.json` + `tool_selection_rules.py` 实现智能工具推荐和状态标记。
- **记忆-执行关联**：记录工具执行历史，避免重复错误（如 POI 扫描工具版本选择）。
- **项目级 MCP 支持**：适配 Trae 新功能，配置可独立于项目。

### 改进
- 重构导入路径，解决相对导入问题。
- 优化启动加载器性能，系统完整性检查更完善。
- 增强快照恢复机制，支持跨会话记忆。

### 修复
- 修复 MCP 服务器重启后代码未重载的问题（需用户手动重启 IDE）。
- 修正工具推荐置信度计算。
