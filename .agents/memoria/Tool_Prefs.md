# Tool Preferences

## Research & Exploration
- **Grep Search:** Use for finding string patterns (`redirect(`, `push(`, `Link href=`).
- **Findstr:** Faster for simple text searches in Windows environments.
- **Directory Listing:** Always deep-peek `app/api/` or `scripts/` to find existing logic before creating scripts.

## Automation
- **Git:** Commit frequentemente com mensagens semânticas (feat, fix, style).
