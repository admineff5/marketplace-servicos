# Project Vision: Marketplace de Estética Professional 🛸💎

**Status**: Active Execution (GSD Mode)
**Objective**: Criar o marketplace de serviços mais fluido e estético do Brasil, com agendamentos automatizados via IA e WhatsApp.

---

## 🎯 Core Goals
- **Experiência Premium**: UI/UX que gere o fator "WOW".
- **Automação Total**: Agendamentos por voz e texto via WhatsApp.
- **Eficiência**: Gestão de tarefas simplificada para o lojista.
- **Segurança**: Verificação robusta via WhatsApp (Registo).

---

## 🏗️ Technical Stack
- **Framework**: Next.js 15 (App Router)
- **Database**: Prisma ORM + PostgreSQL
- **Styling**: Tailwind CSS v4 + Lucide Icons
- **AI Engine**: Google Gemini 2.5-Flash (via @google/genai)
- **WhatsApp**: Baileys (Socket Integration)
