1. **Ativar o Modo GSD**:
   - Analisar a tarefa atual e identificar o caminho mais curto para a entrega.
   - Banir qualquer forma de procrastinação técnica (over-engineering).

2. **Mapeamento de Ação**:
   - Listar os arquivos que serão impactados.
   - Definir a ordem de execução para evitar quebras.

3. // turbo-all
   **Execução Direta**:
   - Implementar todas as mudanças em um único fluxo de pensamento.
   - Resolver todos os lints e erros de tipagem imediatamente.

4. **Verificação GSD**:
   - Rodar `check_list.py` para garantir que nada foi esquecido.
   - Capturar prints/vids para provar a execução (se aplicável).
   - Finalizar com: "GOAL REACHED. NO SURPRISES."
